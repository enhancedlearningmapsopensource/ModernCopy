Steps to install ELM software:

1. Download the zip/rar file
2. Unzip the zip/rar file (linux: "tar xvf {file.rar}")
3. Move the moderncopy folder to the root of the web host (linux: "mv moderncopy /var/www/html/")
4. Remove the zip/rar file (linux "rm {file.rar}")
5. Open a web browser and go to the moderncopy folder. E.g. https://localhost/moderncopy/
6. Fill out the form and click ok.
7. Once the database has been imported, refresh the page to continue.
8. Click "Create Account" to begin using the software.
